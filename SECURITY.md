# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

This project is a bootstrap script for setting up a development machine.
It does not process user data or run as a service.

If you find a security issue in `setup.sh` or any script in this repo,
please open an issue on GitHub or GitVerse.

Do **not** expose API keys or tokens in the repository — this project
requires them as command-line arguments only (never hardcoded).
