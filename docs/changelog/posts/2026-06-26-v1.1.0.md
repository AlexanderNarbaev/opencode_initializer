# v1.1.0 — Ecosystem Expansion

**29 modules, 16 providers, 65+ health checks**

## Major Additions

- **Hardware auto-detection**: multi-vendor GPU (NVIDIA, AMD ROCm, Intel Arc) + NPU (Ryzen AI, Meteor Lake) + Apple Silicon
- **LiteLLM API Gateway**: OpenAI-compatible `/v1` endpoint unifying Ollama, vLLM, SGLang backends with auto-routing
- **SearXNG Web Search**: self-hosted private search engine + sanitizer proxy
- **CI/CD Headless Mode**: `--ci` flag for lightweight OpenCode CLI + essential MCPs
- **15+ LLM providers**: OpenAI, Anthropic, Google, Mistral, Groq, Together, Cohere, Fireworks, Cerebras, Perplexity
- **chezmoi dotfiles** manager for team config sharing
- **Devbox** Nix-based isolated dev environments

## New Modules

`22-webui-service.sh`, `23-just.sh`, `24-websearch.sh`, `25-litellm.sh`, `26-providers.sh`, `27-dotfiles.sh`, `28-devbox.sh`, `22-mise.sh`, `21-rag.sh`

## Key Fixes

- ChromaDB and Ollama systemd services: proper working directories and restart policies
- Open WebUI runs as systemd user service, survives reboots
- Version consistency across all files
- CI test paths fixed for modular architecture
