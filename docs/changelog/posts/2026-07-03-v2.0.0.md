# v2.0.0 — Infrastructure as Code + Isolated Circuit + z.ai GLM-5.2

**2026-07-03** · **38 modules, 24 providers, 350+ test assertions**

Major release featuring Infrastructure as Code, Cockpit TUI, Isolated Circuit Mode, and four new LLM providers.

## Infrastructure as Code

PostgreSQL + Qdrant + Redis + Prometheus + Grafana + MemoryLayer — all provisioned via Docker Compose with a single command. The `30-infra.sh` module generates `~/.config/opencode/infra.yml` and starts all services automatically.

## Cockpit TUI

7-tab terminal UI for server management:

| Tab | Key | Content |
|-----|-----|---------|
| System | F1 | OS, CPU, memory, uptime |
| Plugins | F2 | OpenCode plugin status |
| GPU/Models | F3 | GPU detection, LLM model list |
| Sessions | F4 | Active OpenCode sessions |
| Tasks | F5 | Running background tasks |
| Logs | F6 | System + service logs |
| Infra | F7 | Docker containers, health |

## Isolated Circuit Mode

Air-gapped LLM operation for offline or privacy-sensitive environments:

```bash
bash setup.sh --full --isolated    # Enable during install
dev isolated on                     # Toggle after install
dev isolated status                 # Check state
```

Supports Ollama (:11434), LiteLLM (:4000), vLLM (:8000), SGLang (:30000) with auto-detection.
